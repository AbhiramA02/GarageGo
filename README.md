# GarageGo
I am developing a mobile app that allows users to discover, identify, and collect real-world cars they encounter through in-app photo captures. Using AI-powered vehicle recognition, the app identifies a car’s make, model, and generation, then adds it to a collection with rarity, location, and sighting metadata. Inspired by experiences roadtripping!
